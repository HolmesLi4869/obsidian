# obsidian怎么下载呢？
我相信根据上面的描述，大家还是一头雾水，毕竟哪怕学会了obsidian，也会有一个究极问题：我是谁？我在哪？我该怎么下载obsidian？我该怎么下载obsidian插件？？
接下来我给大家一一解答（不过我不知道你是谁，你在哪，我只知道你在学习obsidian）
- 首先，我们先介绍如何**下载obsidian**：
		在浏览器上直接搜索“obsidian官网[https://obsidian.md](https://obsidian.md/)”，点击直接下载即可；
		![[obsidian官网.png|375]]
- 然后，如何**下载obsidian插件**呢？
		99.999%的可能性，大家打开obsidian后，兴致勃勃地关闭安全模式，准备去第三方插件市场探索插件的时候，打开居然一个插件没有，那是因为这是外网的，需要加速器，but 现在有一个不需要加速器的简单方法：
		方案就是pkmer解决问题，现在听不懂pkmer没关系，接着往下看；	
	1. Obsidian插件pkmer：https://pkmer.cn/products/market/ 
	2. 将这个压缩包解压后，直接拖进在.obsidian文件夹里面的plugins文件夹里面即可。
		- 注意⚠️：
			- .obsidian文件夹里面需要自己新建两个文件,命名为plugins和themes;(千万不能写错!)
			- .obsidian文件夹是隐藏文件夹,
				- 方法一：可通过*everything*搜索助手搜索;
				- 方法二：文件管理器中点击“显示隐藏项”（视频教学里面有）
	3. Shiki实验室：【Obsidian 第三方插件怎么装？怎么用？没有网络也可以！所有疑问统统解答-哔哩哔哩】 [https://b23.tv/IczJQmm](https://b23.tv/IczJQmm)
- 最后，如果你和我用的一样都是平板和手机是iOS，而用的笔记本电脑确实Windows，有相应的同步问题；解决方案如下：
	- 只需要在Windows电脑里面打开Microsoft store，搜索iCloud，如下图，直接下载，登录自己iCloud账户即可；

# obsidian有哪些好用的插件呢？
- **Calendar** : 日历,和日记周记联动;
- Recent files：查看最近打开的文件；
- Quite outline : 层次大纲;
- Memos ：类flomo，记录灵感；
- **Surfing** : 在obsidian里面打开网页;
- **Mind map** : 思维导图;
- **Commander** : 增加按钮;
- **File explorer note count** : 显示文件夹下文件数量;
- **Excalidraw** :  手绘流程图;
- Editing toolbar$\bigstar$ : 类似OneNote,可对文字进行润色;
- Colorful tag : 给标签上颜色,格式为 #某某;
- **Easy typing** :  提高markdown写作的体验;
- Emoji toolbar : 输入emoji表情;
- Icon shortcodes : 插入图标,格式如 :smile;
- Icon folder : 给文件夹上颜色;
- File color : 给文件夹上颜色;
	- Iconic:(多余的)
		提醒各位，插件 colorful tag 和 icon folder 只要軟件更新就會消失，要全部重新設定....要慎用
